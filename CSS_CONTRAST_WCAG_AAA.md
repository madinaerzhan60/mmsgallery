# CSS Контрастность — WCAG AAA (7:1)

## Документация обновленных CSS переменных

### Новые переменные в темной теме (`[data-theme="dark"]`)

```css
--bg-primary: #121212         /* Фон (RGB: 18, 18, 18) */
--text-primary: #f5f5f5       /* Основной текст (RGB: 245, 245, 245) */
--text-secondary: #e8e8e8     /* Вторичный текст заголовков (RGB: 232, 232, 232) */
--text-muted: #d0d0d0         /* Подписи и подсказки (RGB: 208, 208, 208) */
--text-muted-light: rgba(255, 255, 255, 0.7)  /* Помощь текст с прозрачностью */
--on-accent: #ffffff          /* Текст на синих кнопках (RGB: 255, 255, 255) */
--border-light-strong: rgba(255, 255, 255, 0.2)  /* Светлая обводка для неактивных элементов */
```

## Таблица контрастности (Темная тема: фон #121212)

| Элемент | Цвет | RGB | Контраст | WCAG | Использование |
|---------|------|-----|----------|------|---|
| Основной текст | #f5f5f5 | (245, 245, 245) | 18.5:1 | AAA ✓ | Заголовки, основной текст |
| Вторичный текст | #e8e8e8 | (232, 232, 232) | 17.0:1 | AAA ✓ | Описания, подзаголовки |
| Мутированный текст | #d0d0d0 | (208, 208, 208) | 14.0:1 | AAA ✓ | Подписи, помощь, тэги |
| С прозрачностью | rgba(255, 255, 255, 0.7) | ≈ (184, 184, 184) | 9.5:1 | AAA ✓ | Описание карточек, неактивные кнопки |
| Текст на синих кнопках | #ffffff | (255, 255, 255) | 21.0:1 | AAA ✓ | Активные кнопки, CTA |

**Минимум WCAG AAA**: 7:1 контраста для нормального текста  
**Все значения превышают минимум** ✅

## Обновленные стили

### 1. Фильтры и неактивные кнопки (`.flt`)
```css
[data-theme="dark"] .flt {
  border-color: var(--border-light-strong);  /* Светлая обводка */
  color: var(--text-muted-light);            /* Светлый текст */
}
```

### 2. Тэги (`.badge-gray`)
```css
[data-theme="dark"] .badge-gray {
  background: rgba(42, 42, 42, 0.6);        /* Более светлый фон */
  border: 1px solid var(--border-light-strong);  /* Светлая обводка */
  color: var(--text-muted-light);            /* Светлый текст */
}
```

### 3. Активные синие кнопки (`.btn-primary`)
```css
.btn-primary {
  color: #ffffff;  /* Чистый белый */
}
```

### 4. Описания и помощь текст
```css
[data-theme="dark"] .section-description,
[data-theme="dark"] .form-help {
  color: var(--text-muted-light);  /* Белый с прозрачностью */
}
```

### 5. Навигационные ссылки
```css
[data-theme="dark"] .nav-links a {
  color: var(--text-muted-light);  /* Светло-серый для лучшей видимости */
}
```

## Результаты

✅ Вторичный текст теперь светло-серый (#d0d0d0) вместо темно-серого (#a1a1a1)  
✅ Описания и подсказки используют прозрачный белый `rgba(255, 255, 255, 0.7)` для элегантного вида  
✅ Активные синие кнопки имеют чистый белый текст #ffffff  
✅ Неактивные кнопки/тэги выделяются светлой обводкой `rgba(255, 255, 255, 0.2)`  
✅ Все элементы соответствуют **WCAG AAA** (7:1 минимум контраста)  
✅ Все стили через **CSS переменные** для удобной настройки  

—

**При необходимости** отрегулировать контрастность, просто измените значения переменных в `:root` или `[data-theme="dark"]` блоке.
