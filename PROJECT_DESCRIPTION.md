# MMS Gallery — Описание проекта

## О проекте

MMS Gallery — это официальная цифровая платформа студенческих мультимедийных работ SDU University (Suleyman Demirel University) в Алматы, Казахстан. Платформа объединяет функции художественной галереи, профессионального портфолио, социальной сети и витрины для поиска специалистов. Здесь студенты творческих направлений публикуют свои работы, получают обратную связь, накапливают портфолио и находят карьерные возможности.

Слоган проекта: **Not Just a Gallery. It is a Creative Pulse.**

## Архитектура и функционал

### Публичная часть
Главная страница построена как редакционный лендинг с крупной типографикой, чередованием светлых и тёмных секций, бегущей строкой дисциплин и подборкой работ в masonry-сетке. Дальше идут блоки топ-артистов, Open to Work, статистика платформы, описание проекта и форма обратной связи. Футер содержит контакты университета, ссылки на Instagram и TikTok, а также навигацию по основным разделам.

### Галерея
Галерея показывает одобренные работы студентов в masonry-сетке без карточного шума и лишней декоративности. Поддерживаются изображения и видео. При наведении на работу отображается лаконичный оверлей с названием и автором. Клик открывает лайтбокс. Для видео используется превью-кадр с иконкой воспроизведения, а в лайтбоксе открывается HTML5-плеер с управлением.

### Система аккаунтов
Регистрация поддерживает выбор специализации, биографии, курса обучения, LinkedIn и портфолио-ссылок, а также статуса Open to Work. Каждый студент получает публичный профиль со счётчиками работ, лайков и подписчиков. Студенты могут подписываться друг на друга, лайкать работы и оставлять комментарии.

### Личный кабинет студента
Дашборд показывает статистику публикаций, ожидающих модерации, одобренных и отклонённых работ, а также суммарные лайки. Студент может загружать изображения и видео, редактировать профиль, обновлять аватар, переключать статус готовности к работе и добавлять внешние ссылки.

### Hire / Open to Work
Отдельный раздел для работодателей и рекрутёров, где отображаются студенты со статусом Open to Work. Есть фильтрация по специализации и сортировка по лайкам или подписчикам. Карточка студента показывает аватар, профессию, количество работ, лайков и подписчиков, а также ведёт на полный профиль.

### Административная панель
Администратор видит сводную статистику, модерацию работ, список студентов, сообщения обратной связи и действия для управления публикациями. В панели доступны просмотр, пометка как прочитанное и удаление сообщений обратной связи.

### Мультиязычность
Интерфейс поддерживает казахский, русский и английский языки. Выбор языка сохраняется в localStorage. Переводятся навигация, кнопки, статусы и уведомления.

## Дизайн-система

### Философия
Визуальный стиль вдохновлён редакционной эстетикой арт-журналов и музейных сайтов. Это строгий, профессиональный язык, который подчёркивает авторский контент. Основной принцип — никаких мягких карточек, избыточных градиентов и декоративного шума.

### Типографика
Заголовки выполнены шрифтом с засечками, таким как Playfair Display или DM Serif Display. Основной текст и интерфейс используют Space Grotesk и Inter. Метки секций пишутся капсом с широким трекингом и предваряются слешем: `/ FEATURED WORKS`, `/ TOP ARTISTS`, `/ OPEN TO WORK`. Числа и даты подаются как крупные типографические элементы.

### Цвета
Светлая тема использует тёплый офф-вайт и белый фон с почти чёрным текстом. Тёмная тема построена на глубоком чёрном и тёмно-серых поверхностях с светлым текстом. Единственный акцент — электрический синий `#2563eb`, который используется экономно на кнопках, ссылках и отдельных акцентах. По умолчанию включена светлая тема.

### Компоненты
Кнопки имеют скругление 10px, аккуратные границы и мягкий hover с небольшим подъёмом. В тёмных секциях используются светлые outlined-кнопки. Галерейные работы показываются без лишних контейнеров: только превью и метаданные. Профили артистов оформлены с arch-top обрезкой. Навбар закреплён сверху, а мобильное меню и переключатель темы подчинены той же минимальной логике.

### Сетка и отступы
Макет основан на редакционной сетке с крупными отступами между секциями и асимметричными блоками. Masonry-сетка адаптируется под размер экрана. Между секциями используется ритм светлых и тёмных блоков, чтобы страница чувствовалась как журнал, а не как типовой университетский сайт.

## Техническая реализация

### Текущий стек в проекте
- Frontend: HTML, CSS, Vanilla JavaScript
- Backend: Node.js, Express.js
- Авторизация: JWT
- Загрузка файлов: Multer
- База данных: SQLite через встроенный `node:sqlite`
- Хранилище: локальная файловая система в `public/uploads`

### Медиа
- Изображения и видео загружаются раздельно по типу файла
- Для видео сохраняется `thumbnail_url`, `file_type` и `duration_seconds`
- Галерея, личный кабинет и админ-панель используют общие helpers для превью и плеера
- Для существующих записей доступен backfill-скрипт `npm run backfill:media`

### Сценарии запуска
```bash
npm start
npm run seed
npm run seed:force
npm run backfill:media
```
